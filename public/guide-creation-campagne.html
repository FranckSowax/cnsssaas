1 Vue d'ensemble du processus

La creation d'une campagne WhatsApp se fait en deux phases principales : la creation du template de message (soumis a Meta pour approbation), puis la creation et le lancement de la campagne elle-meme.

Pourquoi un template ?

Meta (WhatsApp) exige que tous les messages envoyes en premier (hors conversation active) utilisent un template pre-approuve. Cette verification garantit la qualite et le respect des regles anti-spam. Delai d'approbation : 24 a 48 heures.

2 Creer un template de message

Depuis le menu lateral, cliquez sur Templates puis sur le bouton + Nouveau template.

2.1 Informations de base

ChampDescriptionExemple
Nom techniqueIdentifiant unique, sans espaces ni accents (lettres, chiffres, underscores)bgfi_promo_noel_2026
Nom d'affichageNom visible dans la plateformePromotion Noel 2026
CategorieType de message (affecte la tarification)Marketing / Utilitaire
Categories et tarification

Marketing = 35 FCFA/msg (promotions, offres, relances)

Utilitaire = 11 FCFA/msg (confirmations, alertes, notifications)

Choisissez la bonne categorie car Meta verifie la coherence du contenu.

2.2 En-tete (Header) - Optionnel

L'en-tete apparait en haut du message et enrichit visuellement la communication. Quatre options disponibles :

TypeDescriptionFormat
Aucun Pas d'en-tete, message texte uniquement -
Image Photo affichee en haut du message. Ideal pour les visuels de campagne. JPEG ou PNG. Chemin dans /public (ex: templates/promo.jpeg) ou URL publique.
Video Video avec vignette et bouton play. Ideal pour les reels, temoignages, demos. MP4 uniquement. Chemin dans /public (ex: bgfi-pdg-vision.mp4). Max ~16 MB.
Texte Ligne de texte en gras en haut du message Texte court (60 caracteres max)
Astuce : Video a partir d'un reel Facebook

Pour utiliser un reel Facebook comme en-tete video :

1. Telecharger la video du reel (format MP4)

2. Placer le fichier dans le dossier /public du projet

3. Renseigner le nom du fichier dans le champ (ex: bgfi-pdg-vision.mp4)

4. La video sera uploadee automatiquement vers Meta lors de la creation du template

2.3 Corps du message (Body)

Le texte principal du message. Vous pouvez utiliser des variables pour personnaliser le message :

  • {{1}} = Variable 1 (souvent le prenom ou nom du client)
  • {{2}} = Variable 2 (information specifique : montant, date, etc.)
Exemple de message avec variable

Bonjour {{1}}, BGFI Bank a le plaisir de vous informer de notre nouvelle offre de credit immobilier a taux preferentiel. Contactez votre conseiller pour en beneficier.

2.4 Pied de page (Footer) - Optionnel

Texte discret en bas du message. Utilise generalement pour les mentions legales :

Exemple : "BGFI Bank Gabon | Envoyez STOP pour vous desinscrire"

2.5 Boutons (Buttons) - Optionnel

Jusqu'a 3 boutons peuvent etre ajoutes sous le message. Types disponibles :

TypeDescriptionExemple
Lien URLOuvre un lien web au clicTexte: "Activer mon compte" / URL: https://bgfixstudia.com/
TelephoneLance un appel telephoniqueTexte: "Appeler" / Numero: +24107488716
Reponse rapideEnvoie un message predefined au clicTexte: "En savoir plus"

Cliquez sur + Ajouter un bouton pour chaque bouton souhaite, puis remplissez les champs correspondants.

2.6 Soumettre le template

Une fois tous les champs remplis, cliquez sur Creer le template. Le template est :

  1. Sauvegarde dans la base de donnees locale
  2. Soumis automatiquement a Meta WhatsApp pour approbation
  3. Affiche avec le statut En attente

3 Attendre l'approbation Meta

Apres soumission, Meta examine votre template. Le delai est generalement de 24 a 48 heures.

StatutSignificationAction
PENDING En cours de verification par Meta Patienter. Vous pouvez cliquer sur "Synchroniser" pour verifier.
APPROVED Template approuve et pret a l'emploi Vous pouvez creer une campagne avec ce template.
REJECTED Template refuse par Meta Verifier le motif de rejet, corriger et creer un nouveau template.
Synchronisation des statuts

Pour mettre a jour le statut d'un template, cliquez sur l'icone de synchronisation a cote du template, ou utilisez le bouton Synchroniser tout en haut de la page Templates pour mettre a jour tous les statuts en une fois.

Raisons frequentes de rejet

- Contenu promotionnel dans un template marque "Utilitaire"

- Manque de possibilite de desabonnement (ajoutez STOP dans le footer)

- Contenu trompeur ou non conforme aux politiques Meta

- URL raccourcie ou suspecte dans les boutons

4 Creer la campagne

Une fois le template approuve, allez dans Campagnes puis cliquez sur + Nouvelle campagne.

4.1 Nom et type de campagne

ChampDescription
Nom de la campagneNom descriptif pour identifier la campagne (ex: "Promo Noel 2026 - Broadcast")
TypeMarketing pour les promotions, Notification pour les alertes, Relance pour la reactivation

Cliquez sur Suivant pour passer a l'etape de ciblage.

5 Definir le ciblage

Choisissez a qui envoyer la campagne. Trois options de ciblage disponibles :

OptionDescriptionCas d'usage
Tous les contacts Envoi a l'ensemble de la base de contacts actifs Broadcast general, annonces importantes
Segment Envoi a un segment pre-defini Campagnes ciblees (ex: "Clients Libreville", "Comptes Epargne")
Criteres avances Combinaison de filtres : ville, type de compte, age, engagement, etc. Ciblage sur-mesure sans creer de segment permanent
Recommandation : privilegiez les segments

Les campagnes ciblees via des segments ont de meilleurs taux d'engagement et coutent moins cher. Utilisez la segmentation IA ou les criteres bancaires pour cibler les clients pertinents.

6 Selectionner le template et configurer

6.1 Choix du template

Selectionnez un template approuve dans la liste deroulante. Seuls les templates avec le statut APPROVED sont disponibles. Un apercu du message s'affiche a droite.

6.2 Variables (si applicable)

Si votre template contient des variables ({{1}}, {{2}}, etc.), mappez-les avec les champs contact :

VariableChamp contactExemple de valeur
{{1}}Nom du contact"Jean Dupont"
{{2}}Information personnalisee"votre nouveau pret immobilier"

6.3 Image / Video d'en-tete

Si le template selectionne a un en-tete Image ou Video, l'image/video definie lors de la creation du template sera utilisee automatiquement. L'upload vers WhatsApp se fait une seule fois, puis le meme media est reutilise pour tous les destinataires.

7 Planifier ou lancer

Deux options pour l'envoi :

OptionDescription
Envoi immediat La campagne demarre des la confirmation. Les messages sont envoyes par lots progressifs.
Planifier Choisissez une date et une heure. La campagne se lancera automatiquement au moment programme.

7.1 Confirmation et lancement

Un ecran recapitulatif affiche :

  • Le nom de la campagne et le type
  • Le nombre de destinataires cibles
  • Le template selectionne avec apercu
  • Le mode d'envoi (immediat ou planifie)

Verifiez toutes les informations puis cliquez sur Lancer la campagne.

Attention : action irreversible

Une fois lancee, la campagne ne peut pas etre annulee pour les messages deja envoyes. Les messages sont envoyes par lots, vous pouvez mettre en pause la campagne si necessaire.

9 Bonnes pratiques et conseils

9.1 Contenu du message

  • Gardez les messages courts et directs (2-3 phrases max pour le body)
  • Utilisez les variables pour personnaliser (le prenom augmente l'engagement)
  • Ajoutez toujours "Envoyez STOP pour vous desinscrire" dans le footer
  • Un seul appel a l'action clair par message (1 bouton CTA principal)

9.2 Ciblage et frequence

  • Segmentez votre audience plutot que d'envoyer a toute la base
  • Limitez a 1 campagne marketing par semaine maximum par contact
  • Les messages utilitaires (confirmations, alertes) peuvent etre plus frequents
  • Testez d'abord sur un petit segment avant un envoi massif

9.3 Timing

  • Meilleurs horaires d'envoi : 9h-11h et 14h-16h en semaine
  • Evitez les envois le dimanche et les jours feries
  • Planifiez les campagnes a l'avance pour les lancer au bon moment

9.4 Media (Image / Video)

  • Les images augmentent le taux d'engagement de 20 a 30%
  • Les videos sont ideales pour les messages institutionnels, temoignages et demos
  • Format video : MP4, taille max recommandee : 16 MB
  • Format image : JPEG ou PNG, resolution minimum recommandee : 720x720px

9.5 Gestion des segments

  • Vous pouvez ajouter des contacts a un segment via le bouton "Ajouter" dans le detail du segment
  • Vous pouvez retirer un contact d'un segment en cliquant sur l'icone poubelle a cote du contact
  • Les segments dynamiques se mettent a jour automatiquement selon les criteres definis
  • L'ajout manuel de contacts convertit un segment dynamique en segment statique
Checklist avant lancement

1. Le template est bien APPROVED

2. Le ciblage est correct (verifier le nombre de destinataires)

3. Les variables sont bien mappees (tester l'apercu)

4. Le footer contient la mention de desabonnement

5. L'image ou video d'en-tete est correcte

6. L'horaire d'envoi est optimal
